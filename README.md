# Desarrollo de Sistemas de Inteligencia Artificial (DSIA)

Repositorio personal de trabajos prácticos y notebooks desarrollados durante la cursada de la materia **Desarrollo de Sistemas de Inteligencia Artificial**.

---

## 📋 Descripción

Este repositorio contiene los notebooks, ejercicios y prácticos realizados a lo largo de la materia. El objetivo es documentar el proceso de aprendizaje en programación con Python y el uso de herramientas de análisis de datos aplicadas a la Inteligencia Artificial.

---

## 📁 Estructura del repositorio

```
DSIA/
│
├── notebooks/
│   ├── deepnote/          # Ejercicios realizados en Deepnote
│   ├── python_repaso/     # Notebooks de repaso de Python
│   └── python_librerias/  # Notebooks con librerías de Python (NumPy, Pandas, etc.)
│
└── README.md
```

---

## 🛠️ Tecnologías utilizadas

- **Python 3**
- **Jupyter Notebook / Deepnote**
- Librerías: `NumPy`, `Pandas`, `Matplotlib`, entre otras.

---

## 📚 Contenido

| Carpeta | Descripción |
|---|---|
| `deepnote/` | Notebooks trabajados en la plataforma Deepnote durante las clases |
| `python_repaso/` | Ejercicios de repaso de conceptos fundamentales de Python |
| `python_librerias/` | Práctica con librerías científicas y de análisis de datos |

---

## ▶️ Cómo usar este repositorio

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/FedericoHurtado/DSIA.git
   ```
2. Navegar a la carpeta deseada dentro de `notebooks/`.
3. Abrir los archivos `.ipynb` con Jupyter Notebook, JupyterLab o Deepnote.
